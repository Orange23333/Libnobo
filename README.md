# Libnobo
A software for different groups to audit media with theirselves rule, and control the media access.

# 废话
*Libnobo取自Limit but no block。
*本软件的创作初衷是为了解决部分组织无法轻松完成应各国法律不同导致的媒体审核问题，而我们要解决的具体来说就是如何对接各个国家的审核。
    方法有两个：
    1. 国家或免费组织用该软件来审核它国媒体，并从此提供接口，来访问未经本地审核的外部资源。当然组织（和本软件的配置）自身要经过审核，符合当地标准才行。
    2. 拥有未审核媒体的组织自行建立审核部门，通过本软件自动进行多国审查。
